# Manifests

Manifests are JSON files that define which assets to generate, located in `akonado/manifests/`. They are auto-generated by the pipeline and can also be edited manually.

## Manifest Types Overview

| Manifest | Generated By | Purpose |
|----------|-------------|---------|
| `script.json` | LLM | Script structure (chapters, scenes, character definitions) |
| `characters.json` | LLM | Character sprite prompts and expression definitions |
| `backgrounds.json` | LLM | Background image prompts |
| `bgm.json` | LLM | Background music prompts |
| `se.json` | LLM | Sound effect prompts |
| `voice_config.json` | LLM | TTS voice configuration |
| `dialogue.json` | Auto-extracted | Dialogue data extracted from .ks scripts |

## characters.json

Defines character sprite appearance and expressions.

```json
{
  "type": "characters",
  "items": {
    "girl": {
      "name": "Girl",
      "seed": 12345,
      "size": [768, 1024],
      "base_prompt": "1girl, anime style, long black hair, school uniform...",
      "expressions": {
        "normal": "calm expression, gentle smile",
        "happy": "bright smile, sparkling eyes",
        "sad": "teary eyes, slight frown",
        "angry": "furrowed brows, clenched teeth"
      }
    }
  }
}
```

| Field | Description |
|-------|-------------|
| `name` | Character display name |
| `seed` | ComfyUI seed (consistent appearance; omit for random) |
| `size` | Image dimensions `[width, height]`, default `[768, 1024]` |
| `base_prompt` | Base appearance description (English) |
| `expressions` | Expression variants; key is expression name, value is description |

After generation, each character produces a subdirectory: `assets/characters/<character_id>/`, containing transparent-background PNGs for each expression.

## backgrounds.json

Defines background images.

```json
{
  "type": "backgrounds",
  "size": [1920, 1080],
  "items": {
    "train_station_night": "anime style background, train station at night, rainy...",
    "park_afternoon": "anime style background, sunny park, green trees..."
  }
}
```

| Field | Description |
|-------|-------------|
| `size` | Image dimensions, default `[1920, 1080]` |
| `items` | Key is background ID, value is English scene description |

## bgm.json

Defines background music.

```json
{
  "type": "bgm",
  "format": "mp3",
  "duration": 150,
  "items": {
    "rain_night_mood": "Gentle piano melody with soft rain sounds...",
    "happy_reunion": "Joyful orchestral piece with violin and piano..."
  }
}
```

| Field | Description |
|-------|-------------|
| `format` | Audio format, default `mp3` |
| `duration` | Default duration (seconds), default `150` |
| `items` | Key is BGM ID, value is English music description |

## se.json

Defines sound effects.

```json
{
  "type": "se",
  "format": "mp3",
  "items": {
    "rain_sound": {
      "prompt": "Continuous rain sound, gentle pitter-patter...",
      "duration": 10
    },
    "door_close": {
      "prompt": "Sound of a heavy wooden door closing firmly...",
      "duration": 2
    }
  }
}
```

| Field | Description |
|-------|-------------|
| `items.<id>.prompt` | English sound effect description |
| `items.<id>.duration` | Duration (seconds) |

## voice_config.json

Defines character TTS voice configuration. See [TTS Setup Guide](tts-setup.md#voice-configuration-voice_configjson) for details.

```json
{
  "characters": {
    "girl": {
      "profile": "You are a young girl with a gentle, clear voice.",
      "voices": {
        "mimo": "zh-CN-XiaoyiNeural",
        "qwen": "female-1"
      },
      "instruct_qwen": "A gentle young female voice"
    }
  },
  "emotion_rules": [
    {"label": "happy", "keywords": ["haha", "great"]},
    {"label": "sad", "keywords": ["sorry", "sad"]}
  ],
  "emotion_directions": {
    "neutral": "Say this line in a natural, calm tone.",
    "happy": "Say this line in a cheerful, light tone.",
    "sad": "Say this line in a low, slightly choked tone."
  }
}
```

## dialogue.json

Auto-extracted from .ks scripts. Do not edit manually.

```json
{
  "type": "dialogue",
  "characters": ["girl", "returnee"],
  "lines": [
    {
      "file": "story/chapter01/chapter01_01.ks",
      "chapter": "chapter01",
      "line_no": 14,
      "character": "girl",
      "text": "He said he would come back today...",
      "voiced": true
    }
  ]
}
```

## Management Commands

```bash
# View manifest contents
python -m akonado list              # list all
python -m akonado list characters   # view character manifest

# Clean generated files
python -m akonado clean characters
python -m akonado clean all

# Edit in Web GUI
python -m akonado web
```
